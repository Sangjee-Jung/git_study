터미널 오픈
연동기능을 잘 활용하자, 티커 설정하면 그때부터 자동 연동
티커 로드 한 후 GP MAX 검색 시 도식화된 그래프로 확인이 가능함
	1. 펑션 (재무제표, 차트, 뉴스 기능등)
	2. 티커 ->블룸버그에서 데이터를 보유하고 있는 주체 대상, 주식 뿐 아니라 회사채 등도 티커로 나와있음
		a. Des --> Description: 기본정보 제공, 주식관련 데이터 위주로 제공 ,시총, 최저가, 최고가, per 
			i. 채권 티커로 Des는 특정채권에 대한 정보를 제공. 이자율, 이표, 등급 등
		b. FLDS: 데이터 필드를 검색할수 있는 기능, 상단에 배치된게 주로 많이 보는 정보
		c. HP: RAW 데이터를 시계열로 뽑는 기능
		d. GP: 데이터를 시각화 하는 기능, 우상단 차트편집 기능 활용하기
	3. Research
		a. 블룸버그 자체
			i. BI로 검색 ->  섹터 안으로 들어가면 오른쪽에 주요지표/ 우하단에 주요기업리스트 확인 가능
			ii. 매크로 및 평가를 보면 주요 요인 및 VAL 관련 정보를 볼수 있음
		b. 증권사 리서치
			i. DS로 검색 -> 종목이라는 부분에 특정 기업, 검색이라는 곳에 키워드로 검색 --> 특정기업에 특정 키워드 포함된 리포트 검색가능
		c. FIRS로 주요 키워드가 들어간 뉴스 검색 가능
		d. TREN : 뉴스 트렌드 AI로 분석 , 8시간동안 가장 핫한 순으로 나열
			i. 목록에 KOSPI로 검색시 KOSPI 안에서의 핫한 순으로 나열
	4. 산업
		a. CCB 
			i. 어떤기준으로 산업을 분류할지 --> BICS, GICS 등
			ii. 유사기업 검색 가능 
		b. EQS -> 조건검색, 활용도가 높음
		c. USER -> 시장참여자 설정하면 많이 사용된 기능코드를 볼 수 있음
		d. 제품또는 제품군 관련 회사도 찾을 수 있음. 
			i. 기업개요 --> 제품군 키워드 추가 --> 관련된 기업 추출
		e. SPLC 
			i. 산업 전방 후방 관계를 검색할수 있는 기능
		f. MA
			i. MA 거래 검색 가능
			ii. 시계열 탭을 활용하면 전반적인 특정섹터의 MA 트렌드를 확인 가능
	5. 특정기업
		a. 표준화, 공시기준
			i. 표준화: 블룸버그 standard
			ii. 공시기준: 공시 그대로
		b. ESG, FA
			i. 블룸버그 ESG SCORE 정보 제공
		c. CACS
			i. 기업공시 관련 검색 기능
			ii. 공시 유형, 날짜 필터 가능
		d. ANR
			i. 특정기업에 대한 애널리스트 컨센서스 확인 가능
		e. WACC
			i. WACC 확인
		f. BETA
			i. 베타값 확인
			ii. 상대지수를 변경할 수 있음
			iii. FLDS를 활용하여 고급 검색 가능 --> FLDS에서 기업 선택 후 BETA 설정한 후 기초값이나 상대지수를 직접 변동시켜 바로 확인 가능
	6. 매크로
		a. ECST
			i. 국가별로 DATA 확인 가능
		b. ECFC
			i. 전망치 
		c. ECO
			i. 경제 캘린더
		d. USDKRW --> USDJPY로 검색하면 이종화폐로 검색 가능
		e. FXFC
			i. 환율 전망치
		f. WCRS
			i. 환율 비교 정보 확인가능
	7. 국가
		a. CSDR
			i. 국가 등급 확인 가능
		b. CRP
			i. 특정 국가의 프리미엄
			ii. RF 금리 확인 가능, 10년 벤치 마크
		c. SRSK
			i. 부도등급 확인 가능
BU라는 기능으로 웨비나 서비스 활용가능
